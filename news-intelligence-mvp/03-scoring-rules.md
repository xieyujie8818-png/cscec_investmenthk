# 03 新聞欄位、分類與評分規則

## 新聞登記欄位

每條新聞入庫時至少填寫以下欄位。MVP階段可用Excel、飛書多維表格或其他公司允許的平台承載。


| 欄位                   | 必填  | 說明                  |
| -------------------- | --- | ------------------- |
| news_id              | 是   | 格式建議：YYYYMMDD-001   |
| title                | 是   | 原文標題，可保留英文          |
| publish_date         | 是   | 原文發佈日期              |
| source_name          | 是   | 來源名稱                |
| source_url           | 是   | 原文鏈接                |
| source_credibility   | 是   | A/B/C               |
| region               | 是   | 國家或區域               |
| country              | 否   | 具體國家                |
| sector               | 是   | 交通、能源、房建、投融資、政策、競爭等 |
| topic_tags           | 是   | 關鍵詞5個               |
| digest_section       | 是   | 金融與經濟/建築與地產/北都專題（對外）；可另標項目/拓展（內部） |
| summary              | 是   | 100-200字摘要          |
| business_implication | 是   | 對公司的可能意義            |
| recommended_audience | 是   | 領導/項目/拓展/部門         |
| importance_score     | 是   | 1-5分                |
| relevance_score      | 是   | 1-5分                |
| actionability_score  | 是   | 1-5分                |
| total_score          | 是   | 自動或人工計算             |
| human_status         | 是   | 待審核/已推送/入月報/歸檔/剔除   |
| reviewer             | 否   | 審核人                 |
| review_notes         | 否   | 人工判斷和後續跟進           |


## 分類規則

### 地區分類

首期MVP以香港本地片區和指定專題為主，不再按海外區域分類。若一條新聞涉及多個片區，填主要影響片區，其他片區放入關鍵詞。

- 北都
- 洪水橋
- 新田/河套
- HKU/港大周邊
- 啟德
- 深水埗
- 清水灣
- 九龍東/九龍城
- 新世界/11 SKIES
- 全港/宏觀

### 行業分類

每條新聞選一個主分類，可補充多個標籤。另需填寫日報欄目 `digest_section`，用於控制每日推送配額。

### 日報三欄分配（`digest_section`）

**編輯準則（完整版見 `app/cursoragent/規則.md`）：**

1. **以報道主線定欄**（標題 + 導語首段），不以正文偶爾提及的專題詞決定。
2. **【北都專題】** 須北都／洪水橋／流浮山／北都大學城等為敘事主體；**東九龍綠色集運、啟德** 等非北都基建歸 **建築與地產**。
3. **標題與正文不一致** 時讀導語：例「何文田新校舍動土」但主線為北都大學城 → 歸北都。
4. **工程議價／招標** 若明確以「北都工程」為焦點 → 歸北都，非一般建造欄。
5. **國務院／大灣區跨境制度安排**（如港澳遊艇自由行）→ 歸 **金融與經濟**，不因「遊艇／泊位」誤歸地產。
6. 自動評分見 `daily-brief-app/app/scoring.py`；定稿前須用本準則覆核，候選池標註錯誤時以本準則為準。

- 今日金融與經濟
- 建築與地產
- 金融與經濟
- 建築與地產
- 北都專題
- 項目（內部篩選，可映射至北都專題或建築與地產）
- 拓展（內部篩選）
- 北都產業規劃
- 洪水橋產業園
- HKU學生宿舍
- 啟德綠色智慧交通
- 樓市改建
- 酒店/商廈改學生宿舍
- 新世界財政
- 政策與監管

### 推送對象判斷


| 推送對象  | 適用新聞                          |
| ----- | ----------------------------- |
| 決策層領導 | 北都、啟德、樓市改建、新世界財政中的重大政策、交易、風險或趨勢 |
| 項目負責人 | 指定片區的土地、交通、改建、租金、酒店成交、宿舍需求及項目風險 |
| 拓展負責人 | 可轉化為投資、租賃、改建、合作或項目跟進的線索 |
| 部門同事  | 可作月報素材、知識沉澱或後續研究的新聞           |


## 評分模型

總分滿分100分，用於排序，不代替人工判斷。

```text
總分 = 重要性 x 35% + 相關性 x 35% + 可行動性 x 20% + 來源可信度 x 10%
```

各單項先按1-5分評定，再折算到100分。若使用表格，可按以下公式粗略計算：

```text
total_score = (importance_score * 0.35 + relevance_score * 0.35 + actionability_score * 0.20 + credibility_score * 0.10) * 20
```

發佈日期仍需保留，用於確認新聞來源和每日收集範圍，但不作為獨立評分項。每日推送機制本身已經保證新聞的基本時效性。

## 重要性評分


| 分數  | 標準                            |
| --- | ----------------------------- |
| 5   | 涉及固定專題中的重大政策、重大交易、重大財政風險、重要土地或基建安排 |
| 4   | 涉及指定片區的重要項目、租金/成交趨勢、企業資產處置或政策變化 |
| 3   | 對某一專題有參考價值，但影響範圍有限         |
| 2   | 一般市場新聞，與部門工作弱相關               |
| 1   | 信息價值低或與固定專題幾乎無關               |


## 相關性評分


| 分數  | 標準                       |
| --- | ------------------------ |
| 5   | 直接關聯北都、HKU學生宿舍、啟德交通、樓市改建或新世界財政專題 |
| 4   | 關聯指定片區、指定公司、相關租金/成交或潛在項目線索 |
| 3   | 可作背景參考，但不直接形成工作線索        |
| 2   | 僅泛泛涉及宏觀或行業               |
| 1   | 與部門工作基本無關                |


## 可行動性評分


| 分數  | 標準                |
| --- | ----------------- |
| 5   | 明確需要跟進、匯報、研判或對接   |
| 4   | 可能形成項目、合作、投資或風險線索 |
| 3   | 可支持月報觀點或市場研究      |
| 2   | 主要作背景信息           |
| 1   | 暫無行動價值            |


## 來源可信度分


| 來源等級 | 分數  |
| ---- | --- |
| A    | 5   |
| B    | 4   |
| C    | 2   |


## 推送門檻


| 場景      | 建議門檻             |
| ------- | ---------------- |
| 領導每日快訊  | 總分80分以上，且重要性不低於4 |
| 納入《香港簡訊》 | 總分**70分以上**，去重後，且通過業務審核；定稿總量 **15–20 條** |
| 候選池展示  | 各欄 Top 20，總分70分以上優先 |
| 部門全量素材  | 總分60分以上          |
| 月報素材池   | 總分65分以上，或人工標記為重要 |
| 剔除      | 總分低於50，且無人工保留理由  |


## 去重規則

- 同一事件多個來源報道時，優先保留A級來源。
- 若A級來源信息過短，可保留B級來源作背景，但每日推送只放一條。
- 標題不同但內容高度相同時，合併為同一條新聞，在備註中保留補充鏈接。
- 對涉及重大事項的新聞，至少保留一個官方或公司公告來源。

## 內容類型與金融欄規則

| 規則 | 說明 |
| --- | --- |
| 只要新聞 | 剔除評論、專欄、樓評、社評、筆陣等時事評論稿 |
| 金融欄避開個股 | 恒指/個股升跌短訊、單一股票代號報道（如「騰訊（0700）」）除非同時涉及宏觀或政策 |
| 金融欄優先 | 宏觀經濟數據、財政金融政策、金管局/立法會/政府措施、香港本地政策（不限於金融） |
| 目錄摘要 | 每條附 **一至兩句摘要**，直接取自 **原文首句**（可含第二句），照錄，不做 AI 改寫 |
| 列表採集 | 各報僅讀列表主欄，**禁止**側邊欄、點擊排行、熱門推薦 |
| 標題優先 | **所有列表**先以標題匹配度預篩，僅通過者才抓正文；禁止逐篇無差別爬取正文 |
| HKET 即時新聞 | `https://inews.hket.com/sran001/全部`；標題預篩尤其嚴格 |

關鍵詞配置：`daily-brief-app/config/keywords.yaml`（`opinion_patterns`、`stock_exclude_patterns`、`macro_keywords`、`hk_policy_keywords`）。
