# 01 試點範圍與分類體系



## 試點定位



本MVP服務於中國建築公司香港海外投資部。現階段先不開放同事自行修改新聞摘取範圍，而是採用固定專題清單，由信息整理人每日按既定方向從指定來源摘取新聞。



## 首期固定專題範圍



首期MVP聚焦香港本地宏觀金融與經濟、建築與地產、北都、啟德、樓市改建及重點企業財政風險相關的新聞。暫不做海外泛行業監測。



| 專題 | 關注重點 | 典型線索 |

| --- | --- | --- |

| 今日金融與經濟 | 供決策層快速掌握宏觀環境、金融市場、企業財政及政策背景 | 利率、港股、財政政策、經濟數據、企業融資、重大市場風險 |

| 建築與地產 | 建築、地產、樓市、商業物業、酒店、租金、成交與改建趨勢 | 地產成交、商廈/酒店成交、樓市政策、租金變化、改建項目 |

| 北都：洪水橋產業園 | 洪水橋產業園公司、產業園注資、招商、片區開發、產業用地 | 產業園公司成立或注資、招標、土地或基建安排、片區開發模式 |

| 北都：產業規劃 | 北部都會區產業定位、新田科技城、河套、北都專屬法例、創科產業 | 政府政策、立法會討論、產業規劃、園區公司、基建及融資安排 |

| HKU學生宿舍 | 港大宿舍、城中學舍、酒店/商廈改學生宿舍、港大附近宿舍供應 | 港大發債或宿舍計劃、酒店改裝、宿位供應、學生住宿租金 |

| 啟德：綠色智慧交通 | 啟德智慧綠色集體運輸、啟德交通接駁、啟德體育園及郵輪碼頭交通 | 運輸方案、招標、營運安排、交通配套、區內客流 |

| 樓市：深水埗改建 | 深水埗酒店、商廈、舊樓或工廈改建，特別是學生宿舍或共居用途 | 改裝學生宿舍、補地價/地契安排、收購成交、租金回報 |

| 樓市：清水灣改建 | 清水灣及周邊物業改建、科大相關住宿或教研配套 | 學生宿舍、教研配套、住宅或酒店改建、科大相關需求 |

| 新世界財政 | 新世界資產出售、債務、黑市/市場傳聞、11 SKIES、機管局相關安排 | 拋售、融資、債務重組、商業項目營運、機管局合作或租賃安排 |

## 每日《香港簡訊》欄目（對外定稿格式）



飛書群與完整日報均按以下三欄呈現：



| 欄目 | 建議條數 | 主要讀者 | 內容定位 |

| --- | --- | --- | --- |

| 【金融與經濟】 | 靈活，總量內優先 | 決策層、部門同事 | **宏觀經濟、香港政策**為主；避開個股短訊；只要新聞稿，不要評論 |

| 【建築與地產】 | 靈活，總量內優先 | 決策層、項目/拓展同事 | 新盤、成交、租金、學生宿舍、改建、企業財政相關樓市 |

| 【北都專題】 | 靈活，總量內優先 | 決策層、項目負責人 | 北都產業、大學城、土地、洪水橋、政策與片區開發 |



**定稿總量：** 候選池與正式產出均建議 **15–20 條**；須去重，且內部評分 **≥70 分**（見 `03-scoring-rules.md`）。不再硬性限制各欄固定條數。



## 多日匯總編制



若使用者選取 **多個日期**（如 6月13日–6月15日），系統與 Agent 應將該區間內所有候選新聞 **去重、評分、篩選後，匯總產出同一份定稿**，而非按日拆成多份。



- 候選池與 `selection.json` 以日期區間標識保存（結束日期目錄，或 `YYYY-MM-DD_YYYY-MM-DD` 目錄）。

- 定稿標題顯示日期區間（如「2026年6月13日–15日」）。

- 每條正文【本報M月D日消息】使用該新聞 **原文發佈日期**，非編制日。

## 範圍管理



首期MVP不設同事自助修改新聞摘取範圍。若需要新增或刪除專題，由業務負責人統一確認後再更新本文件和新聞登記表分類。
