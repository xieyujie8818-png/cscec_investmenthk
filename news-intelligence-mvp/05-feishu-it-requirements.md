# 05 飛書與IT對接需求清單

本文件用於與IT同事對接。方案不預設具體實現必須使用某一種飛書能力，IT可根據公司安全、權限和平台規範選擇自定義機器人、企業內部應用或公司既有消息平台。

## 業務目標

每日將經人工審核的 HKET 新聞，編制為 **《香港簡訊》** 並推送至飛書。**眾人推送** 僅含提要、三欄目條目與原文鏈接；**月報素材標記** 僅推送至市場部負責日報/月報的指定同事。

## 推送範圍（6月2日試點後調整）

| 推送包 | 對象 | 頻率 | 模板 | 內容 |
| --- | --- | --- | --- | --- |
| 部門每日簡訊 | 部門飛書群（眾人） | 每工作日 | `daily-feishu-digest.md` | 今日提要；【金融與經濟】【建築與地產】【北都專題】各條標題+一句摘要+原文鏈接；完整版鏈接 |
| 領導完整稿 | 決策層（文檔/郵件/指定群，按部門安排） | 每工作日 | `daily-leadership-brief.md` | 目錄+每條2-3段正文+原文鏈接 |
| 月報素材標記 | 市場部負責日報/月報同事 | 每工作日 | `daily-internal-monthly-markers.md` | 月報候選、跟進、未推送歸檔；**不進大群** |

首期不再按「領導版/項目版/拓展版」拆成三份不同飛書快訊；項目與拓展線索合併在《香港簡訊》三欄目內呈現。

## 建議技術路線

### 階段一：人工發送

- 信息整理人完成 `daily-leadership-brief.md`。
- 推送負責人從定稿生成 `daily-feishu-digest.md` 發群。
- 另將 `daily-internal-monthly-markers.md` 發給市場部指定人（私聊或小群）。
- 用 `templates/pilot-run-log.csv` 記錄推送效果。

### 階段二：飛書機器人半自動推送

- 登記表中標記「已審核可推送」後，按 `audience_type` 生成兩類消息：
  - `department_digest` → 部門群
  - `monthly_marker` → 市場部指定人
- 推送結果回寫新聞登記表。

### 階段三：企業內部應用

- 若需審批流、歷史查詢、素材池導出，建議升級為企業應用。

## 功能需求

| 編號 | 需求 | 優先級 |
| --- | --- | --- |
| F-01 | 支持向部門飛書群推送每日簡訊 | 必須 |
| F-02 | 支持保留原文鏈接並可點擊跳轉 | 必須 |
| F-03 | 支持向指定個人推送月報素材（與群推送分離） | 必須 |
| F-04 | 群推送內容不得包含月報素材標記 | 必須 |
| F-05 | 推送前人工確認，不自動發送未審核內容 | 必須 |
| F-06 | 支持記錄推送時間、對象、新聞ID和狀態 | 必須 |
| F-07 | 支持失敗重試或失敗提示 | 應該 |
| F-08 | 支持月報素材標記導出（僅內部角色） | 應該 |
| F-09 | 支持單條新聞反饋（有用/無關/需跟進） | 可選 |
| F-10 | 支持完整簡訊附件或雲文檔鏈接 | 可選 |

## 權限與安全需求

| 編號 | 需求 | 說明 |
| --- | --- | --- |
| S-01 | 群推送名單與月報標記收件人由部門負責人確認 | 避免錯推 |
| S-02 | 外部鏈接符合公司安全策略 | HKET 鏈接白名單或網關 |
| S-03 | 新聞內容不得包含未授權內部信息 | MVP 以 HKET 公開/會員摘取為主 |
| S-04 | AI 摘要使用公司批准方式 | 不得上傳敏感資料到未批准服務 |
| S-05 | 保留操作和推送日誌 | 便於追溯 |
| S-06 | Webhook、App Secret 由 IT 管理 | 不得寫入業務文檔 |

## 推送數據結構

```json
{
  "digest_date": "YYYY-MM-DD",
  "audience_type": "department_digest | leadership_full | monthly_marker",
  "items": [
    {
      "news_id": "YYYYMMDD-001",
      "section": "金融與經濟 | 建築與地產 | 北都專題",
      "title": "新聞標題",
      "catalog_summary": "目錄用一句話摘要",
      "body_paragraphs": ["段落1", "段落2"],
      "source_label": "香港經濟日報",
      "source_url": "https://",
      "monthly_report_candidate": false
    }
  ]
}
```

說明：`monthly_report_candidate` 僅在 `audience_type` 為 `monthly_marker` 時輸出；`department_digest` 消息體不得包含該字段。

## 飛書群消息展示建議

1. 標題行：【海外投資部 · 香港簡訊】日期
2. 今日提要（1-2句）
3. 分欄列出：標題 — 摘要 — 原文鏈接
4. 文末：完整版文檔鏈接

## 需要IT確認的問題

- 機器人能否同時支持「群推送」與「指定人私聊」？
- 完整簡訊以飛書雲文檔還是附件更合規？
- HKET 鏈接是否需要網關？
- 月報標記收件人飛書 ID 由誰維護？

## 驗收標準

- 部門測試群可收到每日簡訊，且無月報標記內容。
- 市場部指定人可單獨收到月報素材消息。
- 原文鏈接可正常打開或按安全規則跳轉。
- 未審核新聞不會自動發送。
