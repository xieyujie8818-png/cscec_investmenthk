# 大埔黃宜坳 · 第一期高尚住宅基本資料

**項目名稱**：大埔黃宜坳高尚住宅及學生宿舍發展項目（第一期）  
**資料來源**：[pdf/大埔黃宜坳高尚住宅及學生宿舍發展方案 (1).pdf](pdf/)（項目簡報 p.3、12–14、18–19）  
**整理日期**：2026-06-10  
**範圍說明**：本文僅摘錄**第一期高尚住宅**；第二期學生宿舍不在本期市調範圍。

## 一、地塊概況

| 項目 | 內容 |
|------|------|
| 位置 | 新界大埔上黃宜坳（丈量約份第 32 約多個地段） |
| 地盤總面積 | 約 281,322 平方呎 |
| 業主 | 中華有限公司及其關聯公司 |
| 現況規劃地帶 | 綠化地帶（第一期）／鄉村式發展地帶（第二期，本期不涉） |
| 環境 | 北依草山、東臨吐露港；毗鄰大埔滘自然護理區、船灣淡水湖、大帽山郊野公園 |
| 交通 | 鄰近大埔墟站及規劃中白石角站 |

## 二、第一期發展規模

| 項目 | 內容 |
|------|------|
| 發展類型 | 中密度多層高尚住宅（分層住宅單位） |
| 地盤位置 | 地塊東側綠化地帶 |
| 申請地盤總面積 | 約 215,249 平方呎（含新建行車道） |
| 發展地盤面積 | 約 160,161 平方呎 |
| 住宅總樓面面積 | 約 384,384 平方呎 |
| 地積比率 | 2.40 |
| 座數 | 4 幢（T1–T4） |
| 樓層 | 約 11–13 層 |
| 建築高度 | 最高約 36 米 |
| 單位數目 | 約 500 伙 |
| 戶均面積 | 約 769 平方呎 |
| 估計居住人數 | 約 1,400 人 |
| 配套 | 住客會所 |
| 景觀 | 大部分單位可望吐露港及船灣淡水湖 |
| 審批狀態 | 發展方案已提交政府部門審批（簡報日期） |
