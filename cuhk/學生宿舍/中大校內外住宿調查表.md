## 五、校外住宿 — OSA 官方指引與市場對標

### 5.1 OSA 建議的校外租房流程

| 步驟 | 要點 |
|------|------|
| 第一步：做好準備 | 了解 18 區分佈；瀏覽地產網站；設定面積、預算、交通、安全標準；閱讀租賃條例 |
| 第二步：尋找住宅 | 建議委託持牌地產代理；注意按揭物業須承按人同意；公屋不可分租 |
| 第三步：簽訂租約 | 留意租期、按金、租金、免租期、印花稅、分租條款、傢俬清單 |
| 第四步：搬入新居 | 自行開通水電煤（如業主未包）；牢記業主/租客隱含權責 |

### 5.2 OSA 列出的實用參考

| 類別 | 連結 / 機構 |
|------|------------|
| 租賃指南 | 地產代理監管局《非本地學生在香港租屋指南》《安心租屋指南》 |
| 法例 | 差餉物業估價署《業主與租客（綜合）條例》指引 |
| 主要地產代理 | 中原、美聯、利嘉閣等（OSA 頁面有連結） |
| 租約樣本 | 地產代理監管局樣本 1、樣本 2 |

**结论**：中大周边商业化学生宿舍供给严重不足——可核实床位仅约数百个量级，相对本科校外需求 9,000+ 人次/学年，结构性缺口极大；酒店长租可补充但单价显著更高。

**月租来源**：OSA CUHK Student-Exclusive Offers（帝逸/麗豪/帝都/安怡居全港）；迎林 HK01 2025/07；酷點 kudian.org 早鸟价区间。
