- **地點**：新界大埔上黃宜坳（丈量約份第 32 約）
- **第一期**：4 幢 11–13 層高尚住宅，約 500 戶，總住用面積約 384,384 呎
- **第二期**：學生宿舍（西側鄉村式發展地帶）— 本期不做
